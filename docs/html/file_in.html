file_in.read_csv_data(path)[source]¶

Read in csv formatted DEGs

Read csv formatted expression data into a pandas DataFrame for further manipulation. The file should be formatted ‘symbol,logfc,pvalue’ as the first row and corresponding data on following rows

Parameters

path (str) – Full path to the target file

Returns

A dataframe containing data from the file input as ‘path’
